// The Golden Rule
Zigzag ≥ Manhattan ≥ Euclidean — always.
The straight line is the absolute shortest distance between any two points. Every other path is longer or equal. No exceptions.

// real_world_applications

Foundation of Everything

This isn't just geometry. These three formulas quietly run inside the technology you use every day.

🗺️

GPS & Google Maps

Real roads = sum of coordinate segments. But the routing engine uses Euclidean distance as a heuristic to decide which roads to explore first, making it fast.

EUCLIDEAN + PATH SUM
🧠

K-Means Clustering

"Which cluster does this data point belong to?" = find the nearest center. That's Euclidean distance, computed millions of times per training step.

EUCLIDEAN
📦

KNN & Recommendation

Netflix finding similar users, Spotify suggesting songs — Manhattan distance works best when features are independent, like genres and play counts.

MANHATTAN
📉

Neural Network Loss

MSE (Mean Squared Error) is Euclidean distance squared between predictions and ground truth. Every backprop step is literally minimizing a distance.

EUCLIDEAN²
🤖

Robotics & A* Pathfinding

The A* algorithm uses a distance heuristic to estimate cost-to-goal. Manhattan for grid robots, Euclidean for free-space robots. Wrong metric = broken paths.
