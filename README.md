# Learning Artifacts

This repository is a collection of small, UI-based learning artifacts that I use to document and share what I learn. The goal is to turn ideas, concepts, and experiments into simple interactive web pages that are easy to explore and understand.

## Purpose

This repository is mainly for:

- sharing learning experiments in a visual and interactive way
- building small single-page HTML artifacts for different topics
- publishing those artifacts through GitHub Pages for easy access

## Structure

The repository currently includes a folder for each topic or concept. For example:

- distance-metrics/: contains a UI-based artifact related to distance metrics
- webmcp/: contains a working travel-agency demo that exposes search and booking tools to browser agents through WebMCP

Each topic folder typically contains an index.html file, which can be opened directly in a browser or deployed as a GitHub Pages page.

## What to Expect

You will find simple HTML-based demos and examples here. Some of these may include things like distance matrices, visualizations, or other interactive explanations that help make technical ideas easier to understand.

## How to View

To view an artifact locally:

1. Open the relevant folder
2. Open the index.html file in your browser

To publish it online:

1. Enable GitHub Pages for the repository
2. Use the appropriate folder or branch as the published source

This repository is intended to be a growing collection of learning-focused UI artifacts.
